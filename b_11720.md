## 숫자의 합
##### N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.

##### - 입력
##### 첫째 줄에 숫자의 개수 N (1 ≤ N ≤ 100)이 주어진다. 둘째 줄에 숫자 N개가 공백없이 주어진다.

##### - 출력
##### 입력으로 주어진 숫자 N개의 합을 출력한다.

```python
sum_a=0

N = int(input())
a = list(input())

for x in a:
    sum_a += int(x)

print(sum_a)
```

- 공백 없이 쓰인 N개의 숫자를 처음부터 정수의 리스트가 아니라 문자의 리스트로 받으면 편하다.
