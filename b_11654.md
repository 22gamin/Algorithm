## 아스키코드
##### 알파벳 소문자, 대문자, 숫자 0-9중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.

##### - 입력
##### 알파벳 소문자, 대문자, 숫자 0-9 중 하나가 첫째 줄에 주어진다.

##### - 출력
##### 입력으로 주어진 글자의 아스키 코드 값을 출력한다.

##### 문자를 아스키 코드로 -> ord()
##### 숫자를 아스키 코드로 -> chr()

```python
A = input()

if type(A) == int:
    print(chr(A))
elif type(A) == str:
    print(ord(A))
```

- ord()함수   
-> 문자를 아스키 코드로 변환시켜주는 함수


- chr()함수   
-> 숫자를 아스키 코드로 변환시켜주는 함수
